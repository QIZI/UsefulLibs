Detailed Description

desc here

Usage:

BOOL8 varName(b0,...,b7)

varName.b0=[true/false]

...

varName.b7=[true/false]

varName( [index] ) - return bit according to index

varname[ [index] ] - equivalent to varName([index])

varName([index], [true/false]) - set state of bit according to index

varName.getValue(void) - returns uint16_t representation of bits in BOOL16

varName.size(void) - return constant count of bits

varName = [uint8_t] - overloaded operator will make easier to copy data from uint8_t
